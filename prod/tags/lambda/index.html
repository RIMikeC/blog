What Is Serverless?

...and why you should care

What is Serverless? Serverless computing is a service, that runs your code for you on a server that it determines. Although your code runs on a server, the term “serverless” is used as the person who writes the code does not need to know anything about the server. It’s a confusing term. There are servers in a serverless environment, of course there are. However, there is a clear separation of responsibilities between the people who manage the servers and the people who write the code that runs on them.
